Layered Architecture

An education dive into designing layered architecture for applications.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to my small personal side project.

This is simply an application framework maintained by my in order to try and build a solid set of libraries to allow a developer to build well structured applications. I try to make use of MVC, IOC via structuremap, Entity Framework, Rhino Service Bus and so on:

To Check it out simply perform the following in your git bash

$ git clone https://github.com/Foxpips/LayeredArchitecture.git

Background

The main purpose of this project is to allow me to experiment with different nuget packages, experiment with c# and try to build a decent set of core libraries in a sensible manner that can be easily maintained and added to. My main goal is to develop a definitive set of classes, interfaces, structs, services etc that will form a solid base layer for any team starting to write a set of core libraries etc.
